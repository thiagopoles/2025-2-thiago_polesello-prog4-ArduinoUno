## Projeto Arduino Uno – Programação 4

Este repositório contém um projeto desenvolvido durante a disciplina de Programação 4, orientada pelo professor Miguel Debarba, ao longo da primeira metade do 4º semestre do curso.

O projeto tem como objetivo integrar Arduino Uno com uma aplicação web, utilizando PHP, HTML, CSS e banco de dados, permitindo registrar e visualizar dados enviados pelo dispositivo.

A aplicação demonstra conceitos de integração entre hardware e software, comunicação com banco de dados e desenvolvimento de uma interface web para manipulação e visualização das informações coletadas.
